# Hamilton Dynamic User Experience (HDUX)

Welcome to the HDUX component library documentation.

HDUX is a React component framework designed for building dynamic user experiences.

## Getting Started

Install HDUX in your project:

```bash
npm install hdux
```

Import and use components:

```tsx
import { Button } from 'hdux';

function App() {
  return <Button variant="primary">Click me</Button>;
}
```

## Browse Components

Use the sidebar to explore available components. Each component page includes:

- **Interactive demos** — try different props and states
- **API documentation** — full prop tables with types and defaults
